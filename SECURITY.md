# Security Policy

## Reporting a vulnerability

If you believe you've found a security vulnerability in this project,
please report it by emailing **security@jacobwu.org** instead of opening a
public issue.

Include, if possible:
- A description of the issue
- Steps to reproduce
- The impact you expect

I'll acknowledge receipt within a few days and work on a fix as quickly
as practical.

## Scope

In scope: the code in this repository and the services it runs.

Out of scope: third-party dependencies (report those upstream), social
engineering, physical attacks, and denial-of-service testing against
production.
